Uncomplicated Firewall

What is it?

A firewall simply filters what connections can be made to a computer.

How to Install UFW

sudo apt install ufw

Configuring UFW

To enable UFW, run the following:

sudo ufw enable

To check your UFW configuration, run sudo ufw status verbose

It should return the following:

Status: active
Logging: on (low)
Default: deny (incoming), allow (outgoing), disabled (routed)
New profiles: skip
