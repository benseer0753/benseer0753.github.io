Cron

For Linux

Cron is used to schedule tasks in Linux

crontab

crontab is a utility used to check the processes scheduled for different users. for example:

sudo crontab -u root

This checks the scheduled tasks for the root user on the computer. It is important to check here for any persistence mechanisms. Malware likes to put itself in crontab files so it can show back up after being closed.

/etc/crontab is the global crontab. Also check it for malware.

/etc/anacrontab complements the functionality of /etc/crontab on some systems. It works essentially the same, but does not assume that the computer will be always on. Also check there for malware, same as normal cron.
