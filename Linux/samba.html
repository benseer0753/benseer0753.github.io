Samba

For Linux

Samba is Linux's method of interacting with Windows SMB file shares. It has a simple package name, samba, and its configuration file is located at /etc/samba/smb.conf.

The configuration file is broken up into several sections, each surrounded by square brackets. Each section controls a different shared resource on the samba server, except for [global]; configuration there applies to the server as a whole or serves as defaults for other sections.

[global]
