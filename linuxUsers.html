User Management

For Ubuntu and Linux

Ubuntu Settings

The Ubuntu Settings application has a "Users" tab which can be used to easily manage user accounts in a graphical manner. It is more intuitive (In this case) than using the terminal for managing users.

Adding / Removing Users, Changing Passwords, and Giving / Revoking Admin (root) Status

1. Open Ubuntu Settings

You can access the "Settings" application by clicking the grid of white dots located in the bottom-left of your screen, then clicking the gear icon labelled "Settings".

2. Navigating to the User Management Menu

Once the Settings window has been opened, you must navigate to the "Users" item on the left side of the window.

3. The Actual Management

Before you can begin removing/changing passwords, you may have to click the 'Unlock...' button located in the top right of the window. At the password prompt, copy pasting your password from the README is ideal.

Adding / Removing Users

You may add users by clicking the green "Add User..." button located in the window's Titlebar, or manage other users from the list by clicking their Icon. To remove a user, click the user's icon and scroll to the bottom, and click the "Remove User..." button.

Changing User Passwords

You can easily change insecure passwords by letting Ubuntu create a randomly generated password (The gears icon next to the password prompt). Click a User's icon, and click the "Password" box to access the password change prompt.

Terminal User Management

Note: All commands are case-sensitive. Changing a password from the Terminal will not show it while typed, for security.

Adding a new User

For adding a new user, the adduser utility is ideal. Simply run adduser, and the utility will guide you to adding the new user.

Changing a User's Group

To do this, use the usermod utility. It has the following syntax: sudo usermod -g new_group username. Example: sudo usermod -g assistant john changes the group for the user 'john' to 'assistant'.

Changing a User's Password

To change another user's password from the terminal, use the passwd utility. Syntax: sudo passwd username. Example: sudo passwd john changes the password for the user 'john' to whatever password is specified by the User.

Removing a User

To remove a user from the system, use the userdel utility. Syntax: sudo userdel username. Example: sudo userdel john removes 'john' from the system.
