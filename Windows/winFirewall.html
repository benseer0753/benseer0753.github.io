Firewall Setup

For Windows and Windows Server

Usually, all you need to do for the firewall is turn it on. Sometimes, you'll need to go into the more advanced rules to make sure it's configured properly.

Control Panel

The first way to manage the firewall is through Control Panel. In icon view, click on the "Windows Defender Firewall" setting to open its menu. When in category view, click on "System and Security", then on "Windows Defender Firewall".

gpedit.msc goto Windows settings goto security settings goto Windows defender firewall etc. etc. manage it there by clicking the node

Advanced Firewall Management with the Group Policy Editor

Open the MMC, then File > Add/Remove Snap-ins. Add in Windows Firewall with Advaced Security.

Go to Windows Firewall Properties. You will see three tabs: Domain, Private, and Public.

These are the three types of network in Windows. I'm not sure if these are used much in modern windows.

Go into each tab and ensure the firewall is enabled.

Inbound connections should be blocked, outbound should be allowed.

Logging should be set to both Log dropped and Logged Successful Packets

Increase the Log size Limit to 6500
