Firewall Setup

For Windows and Windows Server

Usually, all you need to do for the firewall is turn it on. Sometimes, you'll need to go into the more advanced rules to make sure it's configured properly.

Control Panel

The first way to manage the firewall is through Control Panel. In icon view, click on the "Windows Defender Firewall" setting to open its menu. When in category view, click on "System and Security", then on "Windows Defender Firewall".
