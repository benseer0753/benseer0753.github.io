Windows Miscellaneous

Remote Assistance

Remote Assistance connections must be disabled. The menu can either be opened by running sysdm.cpl or through searching "assistance" in control panel.

Ensure that remote assistancce is disabled, and check the README on whether remote desktop should be permitted.
