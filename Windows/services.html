Services

Some services are not necessary. They should be disabled to reduce the attack surface of the computer.

Remember: the README takes priority over all instructions here. If the README says to keep it, you keep it.

  • ActiveX Installer
  • Adaptive Brightness
  • Application Layer Gateway Service
  • Bluetooth Support Service
  • Certificate Propogation
  • Fax
  • ISS
  • FTP
  • Microsoft FTP Service
  • NetMeeting Remote Desktop Sharing - VoIP
  • Parental Controls
  • Plug and Play
  • Print Spooler
  • Remote Desktop Configuration
  • Remote Desktop Help Session Manager
  • Remote Desktop Services UserMode
  • Remote Registry
  • RIP Listener
  • Routing and Remote Access
  • Server
  • Simple File Sharing
  • SNMP Trap
  • SSD Discovery Service
  • SSDP Discovery
  • TCP/IP NetBIOS Helper
  • Telephony
  • Telnet
  • Universal Plug and Play Device Host
  • UPnP Device Host
  • Windows Messenger Service

Services to Enable

  • Event log

Also make sure to enalbe any services listed in the README.
