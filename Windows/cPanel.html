Control Panel

User Account Control Configuration

Under System and Security > Action Center > Change User Account Control settings

Move the bar all the way up to its maximum setting.

Disabling Dump File Creation

Under System > Advanced > Startup/Recovery > Settings

Set "Write Debugging Information" to "None"

Windows Credentials

Go to User Accounts and Family Safety > Manage you credentials

Delete all listed credentials until there are no entries in any box. This means removing both Windows and Web credentials.

Power Settings

Under System and Security > Power Options

Require a password on wakeup
