Services

Some Windows features are enabled by default, when they are either insecure or unnecessary.

Remember: the README takes priority over all instructions here. If the README says to keep it, you keep it.

Features to Disable

  • Active Directory Services
  • Internet Information Services
  • Media Features
  • Print and Document Services
  • RIP Listener
  • Simple TCP/IP
  • SMB
  • Telnet
  • Work Folders
