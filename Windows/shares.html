Network Shares

Computer Management

You can manage shared files and folders using Computer Management (compmgmt.msc)

From here, you can add and remove shares. The three defaults are IPC$, C$, and ADMIN$. These do not need to be removed. All others, unless specified in the README, can be removed.

Shares with a '$' in them are hidden, and cannot be seen using usual means. They are still accessible, though, and need to be removed just the same.

Using net

Open a command prompt or PowerShell window as administrator.

Select "yes" if a UAC prompt appears.

Type in net share to list all active shares on the computer.

To delete a share, use net share /DELETE name, replacing "name" with the name of the share.
