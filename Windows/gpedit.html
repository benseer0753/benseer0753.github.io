Group Policy

For Windows and Windows Server

The Group Policy Editor (gpedit.msc) is where most of the actual configuration happens on windows computers. In here, you can manage near-on every aspect of the computer. From auditing to Windows Update, there's an endless array of options to edit. Best part, they're all somewhat organized in a handy directory tree, just like files.
