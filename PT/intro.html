Packet Tracer

Initial Configuration

Hostname

hostname(config)#hostname newHostName

Enabling Passwords

User Exec Password

hn(config)#line console 0
hn(config-line)#password password
hn(config-line)#login

Do the same with line vty 0 15 instead of line console 0.

Privileged Exec Password

hostname(config)#enable secret password
hostname(config)#service password-encryption

Banner

hn(config)#banner motd #Put your banner here!#

Configuration Files

To show the current configuration, saved in volatile RAM:

hostname#show running-config

To show the startup configuration, which is in non-volatile memory:

hostname#show startup-config

IP Address

Assign an IP Adress to an Interface

hostname(config)#interface vlan 1
hostname(config-if)#ip address address mask
hostname(config-if)#no shutdown

Show IP information

hostname#show ip interface brief
